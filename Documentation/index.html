Data Flow Overview

1. Kolide Fleet server runs a scheduled query pack on the client machine that already has osquery installed on it.A. The users goes to log into the secured application and Authelia verifies the user via their username and password.
2. The queried data is delivered to the Kolide Fleet server.B. After the user is validated the traffic is redirected the web proxy Traefik.
3. Log file data from Kolide is shipped by Filebeat to Logstash. If there are multiple servers then each would have a Filebeat agent on it.C.Traefik forwards the request to the Swissknife Handler
4. Logstash aggregates the data recieved from Filebeat and sends it to Elasticseach.D.The Swissknife Handler
5. Elasticsearch holds the JSON formatted data under the index: osquery-result-yyyy-mm-dd. The data can be viewed in Kibana.E.
6. The Trust Engine ingests data from elasticsearch and uses the query fields to determine a trust score. The Trust Engine only requestsF.
